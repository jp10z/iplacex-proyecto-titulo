# Sistema de gestión de reservas de servidores

Proyecto de titulo, un sistema de gestión de reservas de servidores web que a los trabajadores de la empresa coordinar el acceso a la infraestructura compartida, visualizando la información de los estados en tiempo real y registrando dichos eventos para consultas posteriores

## Estructura del proyecto

El proyecto se divide en 2 subproyectos, uno para la api backend y otro para la web frontend, a continuación se detallan las carpetas principales.

|Carpeta|Detalle|
|-|-|
|api|En esta carpeta está el proyecto de la API, se usa Python con Flask|
|web|En esta carpeta está el proyecto de la Web, se usa React con typescript|
|database|En esta carpeta he incluído los scripts usados para inicializar la base de datos, además del diagrama base usado, estos scripts están hechos para funcionar con el motor Oracle|
